Build a personal website with the following requirements:

## Tech Stack
- Next.js (App Router) + Tailwind CSS
- Deploy to GitHub Pages
- Blog content as local Markdown files
- No CMS, no database

## Pages & Navigation
Two pages in the nav bar:
1. **Posts** - Overview of all posts ordered by publishing date. Each post shows a thumbnail, title, and short description. Support pinning posts — pinned posts appear above others with a clear visual separation. Posts are written in Markdown and support images and videos.
2. **About Me** - Short bio with a profile picture and links to GitHub, LinkedIn, email, and a CV/resume download.

## Design
- Minimalist and clean with lots of whitespace
- Mobile responsive
- Dark mode / light mode toggle (default: follow system preference)
- Blue-based accent color scheme
- Inspiration references: https://scottlawsonbc.com/posts and https://shane.engineer/
- Simple, modern feel

## Technical Requirements
- Generate static HTML for GitHub Pages
- RSS feed is not needed
- SEO is not needed
- No comments, no analytics, no search
- No tags or categories
